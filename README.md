# Calculator Program in C

## Description
A simple calculator program written in C that performs basic arithmetic operations using switch-case statements.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Menu-driven program

## Technologies Used
- C Programming
- GCC Compiler

## How to Run
1. Clone the repository
2. Compile using: gcc calculator.c -o calculator
3. Run using: ./calculator
